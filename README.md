# Alex Integration For Slack

An [Alex](http://alexjs.com/) bot for slack that uses [slack-node-client](https://github.com/slackhq/node-slack-client).

Set up:

```
npm install
export ALEX_TOKEN="your-bot-key"
node index.js
```
